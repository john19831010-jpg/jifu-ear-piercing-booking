吉富珠寶銀樓 - 穿耳洞預約系統 使用說明書
===============================================

老闆您好！這套「穿耳洞預約系統」是特別為您店內量身打造的。它不需要任何伺服器月費，也沒有繁複的安裝過程。請花 3 分鐘閱讀以下說明，即可輕鬆上手！

-----------------------------------------------
一、 如何打開這個系統？
-----------------------------------------------
1. 請打開您的「ear_piercing_booking」資料夾。
2. 找到一個叫做「index.html」的檔案。
3. 用滑鼠「雙擊兩下」這個檔案，它就會在您的瀏覽器（例如 Chrome、Safari 或 Edge）中打開。
   （您可以將這個網頁加入瀏覽器書籤/收藏夾，方便每天一鍵開啟；也可以將這個網頁的網址傳到店裡的平板或手機上直接使用。）

-----------------------------------------------
二、 顧客如何進行預約？
-----------------------------------------------
1. 顧客打開網頁後，可以瀏覽各種耳針的材質（醫療鋼、鈦、14K金）與保養步驟。
2. 往下滑到「預約表單」，填入姓名、電話、預約日期、時間與想穿的材質。
3. 顧客點擊「確認送出預約」後，網頁會彈出一個「預約送出成功」的視窗。
4. 網頁會自動幫顧客排版好一段「預約確認的 LINE 文字」，顧客點擊「一鍵複製 LINE 文字」按鈕，就可以直接用 LINE 發送給您的店鋪客服確認。

-----------------------------------------------
三、 老闆如何進入「管理後台」？
-----------------------------------------------
1. 在網頁最下方（頁尾處），有一個小字寫著「吉富員工專區」，點擊它。
2. 或是，您直接在瀏覽器網址最後面加上「#admin」（例如：.../index.html#admin）並按下 Enter。
3. 網頁會出現登入畫面，請輸入您的店用密碼：
   🔑 預設密碼：jifu888
4. 輸入密碼後點擊登入，即可進入後台。

-----------------------------------------------
四、 後台能做什麼？
-----------------------------------------------
在後台，您會看到兩個大分頁：

【分頁 1：預約名單管理】
* 查看名單：所有客人填寫的預約都會即時呈現在表格中。系統會自動幫您按照「預約日期」排序。
* 修改狀態：您可以把新進預約從「待確認」改為「已確認」。客人來穿完耳洞後，可以改為「已完成」。
* 填寫備註：您可以直接在表格內輸入這筆預約的備註（例如：已付清、客自備耳針等），系統會自動存檔。
* 手動新增預約：如果有客人是打電話來，或是當場走進店裡預約，您可以在下方的「商家手動新增預約」區直接幫他建檔。
* 📥 匯出 Excel：點擊「匯出 Excel (.xlsx)」按鈕，就可以直接把目前的所有預約資料下載成 Excel 檔案，方便您用 Excel 開啟或存檔備份。

【分頁 2：預約時間設定】
* 自訂營業日與時間：您可以為週一至週日每一天單獨勾選「是否開放預約」，並單獨設定當天的營業時間範圍與時段間隔。
* 💾 儲存設定並記憶：點擊此按鈕後，系統會永遠記住這個設定。前台客人在選時間時，就只能選到您指定的這些時間段。
* 📅 臨時請假與休假設定：在此分頁的下方，您可以選取特定日期（例如師父臨時請假的日期），點選「加入休假日期」。設定後，前台預約系統會自動阻擋客人點選該日期。
* 🕒 個別日期時段調整（時段開關）：若您某天下午有事出門，可以選擇該日期，點擊「載入當天時段」，然後在出現的列表中<b>取消勾選</b>您不在的那些時段，點擊儲存。前台客人在選取該日期時，您取消勾選的時段就會自動隱藏！

五、 如何更換店裡的 Logo 圖片？
-----------------------------------------------
1. 系統預設已為您設計了奢華的「金色珠寶耳環」幾何標誌作為 Logo。
2. 若您想換成店裡的官方 Logo 圖片：
   * 請將您的 Logo 圖片重新命名為「logo.png」（請確保檔名為小寫的 logo.png）。
   * 將這個「logo.png」檔案直接複製並放到「ear_piercing_booking」資料夾中，也就是和「index.html」同一個地方。
   * 重新整理網頁，網頁頂端就會自動換上您店裡的官方 Logo 囉！
   * 同步到 GitHub：若要將新 logo 同步到線上預約網址，請在終端機中執行 `git add logo.png && git commit -m "Add logo" && git push` 即可。

六、 Google 試算表雲端即時同步設定（全新升級！）
-----------------------------------------------
為了防止多台手機、電腦同時預約時時間撞期，本系統全新升級了「Google 試算表雲端同步功能」。設定後，您的預約資料會即時存入您的雲端 Google 試算表中，店面電腦和您個人手機的後台會秒同步！

【Apps Script 雲端橋樑部署 6 步驟】：
1. 雙擊打開您的 Google 雲端硬碟同步檔案：
   📁 /我的雲端硬碟/真愛密碼/吉富珠寶銀樓預約系統資料庫.gsheet (它會自動在瀏覽器中打開試算表)。
2. 在打開的試算表網頁中，確認下方有兩個工作表（分頁），名稱必須命名為：
   * 第一個分頁名稱：Bookings
   * 第二個分頁名稱：Config
3. 點選試算表上方選單的「延伸模組」->「Apps Script」。
4. 將本資料夾裡的「google_apps_script.js」檔案內的程式碼「全部複製」，並「貼上」覆蓋掉 Apps Script 編輯器內原本的所有預設代碼，點擊上方「儲存 (磁碟圖示)」。
5. 點選右上角的「部署」->「新增部署」：
   * 點選左上角選單齒輪圖示，選擇「網頁應用程式 (Web App)」。
   * 說明：填入「吉富預約雲端同步」。
   * 執行身分：選擇「我」(您的 Google 帳號 john19831010@gmail.com)。
   * 誰有權限存取：必須選擇「任何人」(這能讓客人的手機正常讀寫預約)。
   * 點擊「部署」。此時 Google 會要求您授權（點選「授權存取」、選擇您的 Google 帳號、點選左下角「進階/Advanced」、點選最下方「前往吉富預約雲端同步(不安全)」、最後點選「允許/Allow」）。
6. 部署完成後，Google 會給您一串「網頁應用程式網址 (URL)」。複製這段網址：
   * 進入本預約系統的後台 -> 點選「預約時間設定」分頁。
   * 在最上方「Google 試算表雲端同步設定」的輸入框中，貼上這串網址，點擊「💾 儲存並連線雲端」。
   * 看到綠燈「🟢 連線中 (雲端同步)」即代表同步成功！
   * 💡 同步舊資料：點選下方綠色按鈕「📤 將此電腦的預約同步上傳至雲端」，即可把這台電腦上以前累積的預約直接上傳至試算表中，完全不遺失歷史預約！

七、 注意事項與資料安全
-----------------------------------------------
1. 多端防重複預約：連線試算表後，客人預約時會即時檢查雲端，重複時段將無法點選，同時段只能預約一人的防重疊機制在多台手機與電腦間全面生效。
2. 離線降級保護：若您的店內網路中斷，或 Google 雲端服務異常，系統會自動切換為本地「單機模式」運作，確保客人仍能送出預約，待連線恢復後再點擊同步。
3. 輕鬆管理：除了在 Google 試算表直接查看、編輯與管理資料外，您依然可以隨時在後台點選「匯出 Excel (.xlsx)」下載檔案到本機備份。
